## 1. 핵심 아이디어

**애널리스트 이익전망 상향 모멘텀(earnings estimate revision momentum)** 은 학계에 수십 년간
문서화된 이상현상입니다 — 컨센서스 EPS가 상향되면 주가는 그 정보를 한 번에 반영하지 않고
수주~수개월에 걸쳐 천천히 따라갑니다(post-revision drift). 기관(Zacks, StarMine 등)은 이
신호로 사업을 하지만, 개인 스케일의 자동화 구현은 드뭅니다 — 컨센서스 데이터의 수집·정제가
지저분하고, 두 시장을 공정하게 결합하는 눈금 문제가 까다롭기 때문입니다.

이 시스템의 순위 신호는 단 하나입니다:

```
rev90 = (현재 NTM EPS 컨센서스) ÷ (90일 전 NTM EPS 컨센서스) − 1
      = "전문가들이 석 달간 이익 전망을 몇 % 올렸나"
```

- **NTM EPS** = 향후 12개월 예상 주당순이익. 당해/차기 회계연도 추정치를 잔여 개월수로
  시간가중 블렌딩해 합성합니다 (회계연도 경계 왜곡 제거).
- 가격·차트는 순위에 쓰지 않습니다. 가격의 역할은 관문(자격 검사)에서 끝납니다.

## 2. 전략 규칙 (현재 운영본)

### 유니버스 → 관문 → 순위 → 포트폴리오

```
┌─ 유니버스 (수집 범위 — 수집 단계 필터 없음)
│    미국: S&P500+400+나스닥100 + 전 거래소 시총 $5B+ ≈ 1,290종목
│    한국: 시총 1,000억 원 이상 ≈ 1,430종목 (컨센 존재 ~340)
│
├─ 관문 (매수 자격 — 하나라도 걸리면 순위 무관 탈락)
│    ① 전망 안 꺾임: 90일 궤적 4구간(90→60→30→7→현재) 전부 상향 (min_seg ≥ 0)
│    ② 유동성: 일평균 거래대금 미국 $1B+ / 한국 $0.3B+
│    ③ 가격 상한: 선행PER ≤ 30 (예상이익 대비 과열 차단)
│    ④ 이익 성장: 올해 예상이익 ≥ 작년 실적의 1.5배 (gap ≥ 1.5, 데이터 공백은 통과)
│    ⑤ 품질: 저커버리지(<3~5명)·영업마진<5%·FCF와 ROE 동시 음수·원자재/비성장 업종 제외
│
├─ 순위 (관문 통과자만)
│    rev90 → 자기 시장 전체 분모(미국 1,290 / 한국 ~340, 30일 트레일링 유니온) 내
│    백분위로 환산 → 두 시장 결합 정렬
│
└─ 포트폴리오
     TOP 5 · 각 20% 균등비중 · 5거래일(≈매주)마다 재심사·교체
     carryover / 슬롯 고정 / 손절 / 트레일링스탑 없음 — 교체는 순위와 관문이 전부
```

## 3. 왜 시장별 백분위인가

한국과 미국의 컨센서스 상향폭은 **눈금이 다릅니다**. 실측(2026-07)으로 뜨거운 구간(상위
10~1% 지점)에서 한국이 미국의 약 2배 — 절대값으로 결합하면 한국 종목이 시스템적으로
부풀려져 TOP5를 지배합니다.

그래서 각 종목을 **"자기 시장 안에서 상위 몇 %인가"** 로 환산해 결합합니다:

- 한국 +64% 상향 = 한국에서 상위 ~9% ← 한국에선 종종 있는 성적
- 미국 +42% 상향 = 미국에서 상위 ~5% ← 미국에선 희귀한 성적 → **미국 쪽이 위**

이 선택은 다섯 겹으로 검증됐습니다: 분모 소표본·수집 붕괴 교정, 분모 대칭화(양국 무필터),
체급(시총 컷 사다리) 민감도, 분포 EDA(백분위·robust-z·log-z·상수배율 4개 잣대 수렴),
robust-z 교차 확인 — **모든 합리적 잣대에서 같은 TOP5가 나옵니다.** 절대값 결합과
robust-z 순위는 도전자 트랙으로 매일 병기 기록되며, 판정일에 forward 수익으로 심판합니다.

## 4. 방어 구조 — 3층

| 층 | 감시 대상 | 신호 | 발동 시 |
|---|---|---|---|
| **개별 종목** | 이익전망 궤적 | 4구간 중 하나라도 하향 (min_seg < 0) | 즉시 자격 상실 → 다음 교체일 탈락 |
| **테마 (메모리)** | 한·미 메모리 대표 6종목 (샌디스크·마이크론·WDC·씨게이트·삼성전자·SK하이닉스) | 6개 중 3개가 자기 90일선 아래 (즉시 발동 / 해제 5일 확인) | 메모리 종목 전량 매도 → 현금 권고. 10년 검증: 큰 하락 5번 전부 감지, 최악 손실 −54%→−31% |
| **시장 전체** | S&P500 vs 200일선 (15일 확인) / VIX > 36 (2일) / 하이일드 신용스프레드 급등 | 하나라도 확정 | **전량 현금** 지시. 26~30년 검증: 닷컴·2008·코로나·2022 전부 포착 |

한국 국면(KOSPI 20/80일선 크로스 + 5일 확인, 한국 시스템 7.4년 검증)은 현재 **참고
표시**로 병기되며, "한국 방어 시 한국 종목만 현금"(시장별 스코프) 매매 반영은 검증 후
도입 예정입니다.

가격 손절이 없는 이유: 이 시스템의 매도 신호는 가격이 아니라 **전문가 시선의 철회**입니다.
가격 하락은 오히려 선행PER 관문에 여유를 만들고, 전망이 유지되는 한 보유합니다.
(백테스트에서 타이트 손절은 일관되게 성과를 깎았습니다 — 승자의 휩쏘를 자르기 때문.)

## 5. 시스템 아키텍처

```
06:15 KST  美 컨센서스 수집 (GitHub Actions, 평일)
           yfinance ~1,290종목 → SQLite(ntm_screening) → git push
                                        │
16:30 KST  韓 컨센서스 수집 (로컬 스케줄러, 평일)
           yfinance ~1,430종목 시도 → KR DB → git push  (발송 없음, 수집 전용)
                                        │
18:15 KST  ★ 통합 신호 (GitHub Actions, 평일) ── 유일한 매매 메시지
           美 DB + 韓 DB(sparse checkout) → 관문 → 백분위 결합 → TOP5
           → 국면/감시등/신용·변동성 판정 → AI 종목 브리핑·시황 (Gemini, 폴백 체인)
           → 텔레그램 발송 (채널 + 개인봇) → 원장(unified_vm_log.csv) 커밋
```

- **원장은 append-only, 작성자는 Actions 단일.** 일회성/샘플 실행은 `UNIFIED_NO_LOG=1`.
- 누적 성과는 원장 리플레이로 재구성 — 지시와 성과가 같은 시계(미국 거래일 그리드)를 씁니다.
- 데이터 품질 이상(한국 수집 부실, 백분위 폴백 등)은 메시지 상단에 자동 경고. **조용한
  실패 경로를 남기지 않는 것**이 설계 원칙입니다.

## 6. 검증 철학

이 저장소의 절반은 전략 코드가 아니라 **검증 흔적**(`research/`)입니다. 규칙 하나를
바꾸려면 통과해야 하는 관문:

1. **Leave-One-Winner-Out (LOWO)** — 최대 승자 1~2종목을 빼고도 개선이 남는가?
   (소표본 백테스트의 edge는 대부분 슈퍼위너 1종목 착시)
2. **워크포워드** — 기간을 쪼개 train 승자가 test에서도 이기는가?
3. **인접 안정성** — 파라미터 ±1칸이 고원(plateau)인가, 고립 스파이크인가?
4. **BT == production 정합** — 백테스트 하니스가 실제 운영 코드와 같은 동작을 하는가?
   (시뮬레이터 결함으로 alpha가 7배 부풀려진 사고 이후 필수 절차)
5. **판정일 (forward adjudication)** — 배포 후에도 도전자 변형(절대값 결합, robust-z,
   테마캡 등)을 매일 병기 기록하고, 실현 수익으로 본선을 재심사

그리고 문화적 규칙 하나: **재량 개입 금지.** 룰을 따르거나, (검증으로) 룰을 바꾸거나 —
둘 사이의 임의 개입이 최악입니다.

## 9. 정직한 한계

- **어린 실전 기록**: 통합 트랙은 2026-07 가동 — 강세장만 경험했고, 약세장 방어(국면
  오버레이·감시등)는 과거 지수 데이터 검증이지 이 전략의 실보유 기록이 아닙니다.
- **백테스트는 상한**: 컨센서스 히스토리 제약으로 일부 검증은 look-ahead proxy 상한입니다.
  실전 기대치는 백테스트 수치보다 한참 겸손하게 잡습니다.
- **집중 포트폴리오**: 5종목, 그중 메모리 반도체가 다수인 국면이 있습니다. 테마 신호등이
  감시하지만 실전 발동 이력은 아직 없습니다.
- **데이터 의존**: 컨센서스 소스(yfinance)의 커버리지·안정성 한계. 품질 경고와 폴백을
  깔아뒀지만 근본 개선(국내 컨센서스 소스 이관)은 진행 중인 과제입니다.
- **투자 조언이 아닙니다.** 이 저장소는 개인 연구·자동화 기록이며, 모든 투자 판단과
  책임은 본인에게 있습니다.
