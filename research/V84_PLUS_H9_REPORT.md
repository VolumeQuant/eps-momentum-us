**일시**: 2026-06-01
**기간**: 2026-02-12 ~ 2026-05-29 (74거래일, 100% boost regime)
**Simulator**: entry_fixed total_cash pool, 100 seeds × 3 samples paired BT

## TL;DR

| 결론 | 내용 |
|---|---|
| **H9 = 정직한 trade-off** | Return -17%p 손실 ↔ MDD +7%p 개선, Sharpe +0.54, Calmar +21% |

| **권장** | 강세장 한정 표본 ❌ 즉시 채택 X. 약세장 검증 필수. |

## 3. Robustness 1: Leave-one-winner-out

**v84의 강세장 alpha가 단일 winner 의존인지 확인.**

| 제외 종목 | v84 return / MDD | v84+H9 return / MDD | H9 lift | wins |
|---|---|---|---|---|
| NONE (전체) | +108.4% / -23.9% | +91.2% / -16.6% | **-17.2%p** | 21/100 |
| MU 제외 | +95.7% / -20.9% | +90.1% / -16.6% | -5.7%p | 37/100 |
| SNDK 제외 | +94.6% / -30.3% | +80.8% / -35.4% | -13.8%p | 12/100 |
| BE 제외 | +99.1% / -23.9% | +90.2% / -16.6% | -8.9%p | 26/100 |
| AEIS 제외 | +121.3% / -23.9% | +91.2% / -16.6% | -30.1%p | 3/100 |
| **MU+SNDK 제외** | **+58.5% / -28.7%** | **+56.8% / -22.7%** | **-1.7%p** | **44/100 ≈ 동등** |
| top4 모두 제외 | +63.6% / -28.7% | +55.3% / -22.7% | -8.3%p | 11/100 |

**해석:**
- 전체 -17.2%p 손실은 큰 차이로 보이지만, **MU+SNDK 두 종목만 빼면 -1.7%p (거의 동전던지기)**
- v84의 강세장 우위 = 1-2개 super winner가 견인 (single-stock illusion 패턴 [[feedback_leave_one_winner_out]])
- **MDD는 거의 모든 leave-out에서 H9 우월** — robust한 MDD 개선

→ **return 우위는 winner-dependent (취약), MDD 우위는 robust (안정).**

## 4. Robustness 2: 시기 분할 — ★ 가장 큰 발견

| 기간 | v84 return / MDD | v84+H9 return / MDD | H9 lift | wins |
|---|---|---|---|---|
| **Early half** (2/12 ~ 4/7) | +31.2% / -23.9% | +11.0% / -16.6% | **-20.2%p** | **1/100** ❌ |
| **Late half** (4/8 ~ 5/29) | +43.3% / -14.7% | +49.2% / -14.7% | **+5.8%p** | **100/100** ✅ |

→ **두 기간에서 H9 효과가 완전 정반대!**

**의미:**
- Early (2-3월): 시장이 횡보~약상승, H9가 너무 보수적 → 진입 막아서 손해
- Late (4-5월): MU+SNDK 대박 발생, H9가 그 발생을 catch (가격 이미 +20일 상승 중) → 더 잘 잡음

→ **H9 효과는 시장 모멘텀 환경에 dependent**. robust하지 않음. **다른 시기에서는 또 반대 결과 가능.**
