# v81 종합 보고서 (2026-05-18): MA120 (MA60 fallback) → MA20 단기 모멘텀 필터

## 핵심 요약 (TL;DR)

**production 매수 후보 필터의 MA120 (MA60 fallback)를 MA20으로 변경**. BT 알파 +27.57%p (random 100/0), MDD -18.18→-10.35% 개선. 알파의 본질은 매수 필터가 아니라 매도 트리거 — "MA20 이탈 시 part2_rank NULL → rank>10 즉시 매도" 효과. entry/exit/slots는 (3,10,3) 유지, (2,10,2)는 v82로 분리.

---

## 1. 발단

사용자 질문: "필터가 현재 MA120선 이탈인데 MA60선이나 다른 선이 더 성과 좋은건 아닌지 검증해봐."

추가 발견:
- production이 `price > MA120 (NULL이면 MA60 fallback)` 사용
- 초기 일자에 MA120이 NULL인 종목이 많아 **사실상 MA60 필터로 운영 중**이었음 (BT 검증)
- 그러면 진짜 비교는 "MA60 ↔ MA20"

### 검증 3: 충격 회피 가설 검증 (Early/Mid/Late 분해)
3/30 이란 전쟁 충격 회피가 lift의 주범인가?
```
Early (2/12~3/13, 충격 포함):  +17.60%p, 20/21 wins
Mid (3/16~3/27, 충격 직전 진입): +28.79%p, 10/10 wins

```
**충격 없는 Late 구간이 lift가 가장 큼 → single-event alpha 가설 정확히 반박**

### 검증 4: A vs B 매도 분리 (★ 핵심)
```
Option A (MA20 매수 필터 + 사실상 매도 트리거): +27.57%p, 100/0
Option B (매수만 MA20, 보유는 rank 기준):      -0.29%p, 50/50 (효과 없음)
```
**알파의 거의 100%가 매도 트리거에서 옴** — MA20 이탈 시 part2_rank NULL → rank>10 즉시 매도. 매수 필터 자체는 효과 0.

## 3. 알파 메커니즘

```
MA120 (production) 사용 시:
  - 가격이 MA120 아래로 갔어도 part2_rank Top 30 안에 들어감
  - 보유 종목이 단기 모멘텀 잃어도 rank 안 떨어지면 안 팔림
  - 결과: 큰 손실 후 cut → MDD 큼 (-18.18%)

MA20 (v81) 사용 시:
  - 가격이 MA20 아래로 가면 즉시 part2_rank NULL
  - 보유 종목이 단기 모멘텀 잃으면 즉시 rank>10 트리거 → 매도
  - 결과: "단기 모멘텀 잃은 종목 빠른 cut" → MDD 작음 (-10.35%)
```

이게 매수 진입 필터가 아니라 매도 가속 효과인 이유.
