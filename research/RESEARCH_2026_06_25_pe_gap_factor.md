# PE_gap (trailing PE ÷ forward 12m PE) 팩터 연구 — 사용자 가설 검증 (2026-06-25)

> 발단: 사용자 관찰 — "①trailing PER과 fwd 12m PER 차이(=gap)가 클수록(PER높고 fwd낮음) 잘 오르고, gap≈1이면 지지부진. ②특히 trailing PER<20 + gap≈1이면 우하향." SK하이닉스·삼성 예시.
> 정의: **gap = trailing_PE / forward_PE = forward_EPS / trailing_EPS = 시장이 기대하는 EPS 성장배수.**
> 데이터: DB ntm_current(forward EPS) + yfinance 분기실적 PIT trailing EPS(보고지연+45일) + yfinance 가격. top30 유니버스 91거래일. 재현 스크립트: scratchpad/pegap_*.py.

---

## ⭐ 결론

**가설 ①은 강하게 맞다(IC +0.23, robust). 가설 ②는 부분적(gap이 핵심이지 절대PE는 아님).**
**단 — 이 신호는 우리 2슬롯 시스템엔 안 맞는다**(넣으면 BT −32~−83p, LOWO 기각). 우리 EPS-랭크가 이미 동급 정보를 갖고 있고, 2슬롯 집중은 cross-sectional 팩터를 수확 못 함. **gap이 빛나는 곳은 분산 슬리브(10~15종목, 롱숏 스프레드 +27~51%p) = 별개 상품.**

## 1. 가설 ① 검증 — big gap → 오른다: ✅ 강하게 맞음

| 검증 | 결과 | 판정 |
|---|---|---|
| Cross-sectional IC (gap vs 향후20d) | **+0.219** (90일 중 71일 양수) | ✅ 강함 |
| **PIT trailing EPS로 재검증** (look-ahead 제거) | **+0.232** (오히려 ↑) | ✅ look-ahead 아님 |
| 5분위 향후20d (PIT) | Q1 **+3.5%** → Q5 **+13.9%** 단조 | ✅ 깨끗 |
| 사용자 직접 구간 | gap≈1: **+2.9%** / gap 2.5+: **+12.8%** | ✅ 관찰 일치 |
| 모멘텀 직교성 (corr +0.14, 통제후 IC) | 저모멘텀 +0.21 / 고모멘텀 +0.21 | ✅ 모멘텀 재탕 아님 |
| **현 시스템(part2_rank) 직교성** | corr **−0.06** (거의 무관) | ✅ 새 정보 |
| 시기분할 | 전반 +0.33 / 후반 +0.11 | ✅ 둘다 양수 |

→ **사용자 가설 ①은 엄밀하게 확인됨.** 이번 멀티세션 통틀어 가장 강하고 robust한 단일 신호. yfinance가 KR(삼성·SK하이닉스) trailing PE를 안 줘서 그 예시는 직접 못 봤으나, US 데이터로 패턴 입증.

## 2. 가설 ② 검증 — 저PE+저gap = 우하향: 🟡 부분적 (gap이 핵심, 절대PE는 부차)

2×2 (향후20d 평균):

| | gap≈1(<1.3) | gap 큼(≥1.3) |
|---|---|---|
| **trailing PE<20** | +4.2% | +3.5% |
| **trailing PE≥20** | **+1.8%(최악)** | **+10.8%(최선)** |

- **"지지부진" = gap이 작은 것** (저gap +1.8~4.2% << 고gap +10.8%). ✅ — 단 **작동변수는 gap이지 절대 PE가 아님.** 최악 칸은 사용자가 콕집은 저PE가 아니라 **고PE+저gap**(+1.8%).
- 사용자 콕집 칸(trailing PE<20 & gap<1.2): +6.7%(중앙 +5.9%, 음수 **19%**) — 전체(+8.8%, 음수 31%)보다 **수익은 낮지만(지지부진 ✅) 우하향은 아님**(오히려 음수비율 더 낮음). ❌ 우하향 미확인.
- 해석: 강세장 + EPS상향 유니버스라 진짜 value-trap 우하향은 표본 밖. **"저PE+저gap 우하향"은 broad market/약세장에선 맞을 수 있으나 이 데이터론 미확인.** 확실한 건 **"gap≈1 = 지지부진"**(절대 PE 무관).

## 3. ★ 그런데 왜 우리 시스템엔 안 맞나 — 넣으면 망가짐

진입 종목선택에 gap 적용 (청산룰 고정, PIT gap):

| 변형 | 누적 | vs base | LOWO |
|---|---|---|---|
| baseline (rank) | +216.7% | — | — |
| gap정렬 pool5 | +185% | −32p | — |
| **gap정렬 pool8** | **+134%** | **−83p** | **worst −86p ❌** |
| rank+gap필터≥1.3 | +177% | −40p | — |
| rank+gap z-blend (lam0.3) | +220.6% | +3.8p but **MDD −26.7%(악화)** | worst **+0.0p** null |

**원인 (정밀 규명):**
1. 우리 winner들은 오히려 **고gap**(SNDK 5.83·MU 4.86·LITE 4.01) — gap과 winner가 상충하는 게 아님.
2. **pure gap정렬은 극단 tail(gap 8~11, speculative)을 쫓아** 안정적 winner를 놓침. gap-top(+61%) ≈ rank-top(+64%)이라 **rank보다 낫지도 않음.**
3. **2슬롯 집중은 cross-sectional 팩터를 수확 못 함** — IC는 분산 포트폴리오 속성, 2종목엔 특정 픽 리스크가 지배. LOWO −86p가 확증.

## 4. ★ gap이 빛나는 곳 — 분산 슬리브 (별개 상품)

매일 top-N 동일가중 (비용무시, 팩터 수확 입증):

| N | gap상위 | rank상위 | gap하위 | 롱숏 스프레드 |
|---|---|---|---|---|
| 10 | +61%(MDD−18) | +64%(MDD−15) | +10% | **+51%p** |
| 15 | +60%(MDD−15) | +53% | +33% | **+27%p** |

시기분할 스프레드: 전반 +20%p / 후반 +25%p (둘다 robust).

→ **gap은 10~15종목 분산 슬리브에서 진짜 알파**(롱숏 +27~51%p). 단 gap상위 ≈ rank상위라 **EPS-모멘텀 대비 우월하진 않고 parallel**. 별개의 "PEG-value 성장" 슬리브로는 의미, 현 2슬롯엔 부적합.

## 5. 한계 / 권고
