# 🇰🇷 KR 핸드오프 — 별도 가치(gap) sleeve 제안 (US 발신, 2026-06-26)

## 한 줄 요약
US에서 "기대성장(gap) 분산 sleeve"를 만들어보니 **모멘텀과 거의 같은 성과(둘 다 forward EPS를 읽음)**가 나왔고, **US에선 시너지가 0**(gap·모멘텀 corr +0.83, 좁은 테크 유니버스라 같은 종목). **그런데 KR은 gap이 모멘텀과 직교(corr +0.015)라 — KR에선 별도 가치 sleeve가 진짜 포트폴리오 시너지를 낼 가능성이 높다.** KR이 검증해볼 가치 있음.

---

## 1. US에서 발견한 것

- **gap = NTM(컨센 선행)EPS / TTM 실적EPS = trailing PER / forward PER = 시장 기대성장.** (KR "확신가중"의 그 신호와 동일.)
- US에서 이걸 **딸깍 한 줄 sleeve**로 만듦: eligible 중 **gap 상위 7종목 동일가중, 월 1회 리밸, 약세장이면 현금.**
- 성과: 8년 Calmar **~2.0** vs 모멘텀 **~2.7** = **단순 sleeve가 복잡한 모멘텀의 ~75%를 한 줄로 재현.**
  - 교훈: 알파의 대부분은 단순 팩터(forward EPS)에 있고, 모멘텀의 복잡한 산식은 마지막 25%(robustness·일관성·함정회피)를 사는 것. → **단순 버전이 대부분 재현 = 알파가 진짜(과적합 아님)라는 역설적 증거.**

## 2. 왜 US에선 시너지가 0인가
- gap(레벨)과 모멘텀(revision=변화)은 **같은 책(forward EPS)을 읽음** → US 좁은 테크 유니버스(~137종목)에선 **corr +0.83**.
- 둘 다 "고성장 반도체 사라"고 외침 → 합치면 **같은 베팅 2배(집중)지 분산이 아님.** 융합 8가지(점수블렌드·보너스·슬롯·conviction 등) 전부 실패.

## 3. ★KR 함의 — 여기선 다를 것
- **KR은 broad ~1900 유니버스라 gap이 모멘텀과 corr +0.015 (직교, 독립 IC +0.170)** — 이미 KR이 측정한 값.
- 직교 = **별도 가치 sleeve가 모멘텀과 다른 종목을 들고 → 진짜 포트폴리오 분산 → 시너지(결합 Calmar↑·MDD↓) 가능성.**
- 이건 **KR이 지금 하는 "확신가중"(보유 3종목 reweight, sizing 오버레이)과 다른 축**:
  - 확신가중 = **같은 3종목** 비중조절 (새 종목 X)
  - 별도 sleeve = **모멘텀이 안 든 고gap 종목**까지 새로 매수 (포트 레벨 분산)
- → KR은 이 **별도 sleeve를 추가로 테스트**해볼 가치 있음.

## 4. 제안 sleeve 스펙 (US 설계 차용, KR 재튜닝 필요)
| 항목 | US 값 | KR 조정 |
|---|---|---|
| 신호 | gap = 컨센 선행EPS / TTM | 동일 (FnGuide 컨센) |
| 종목 | gap 상위 K=7 동일가중 | K 재스윕 (KR 유니버스) |
| trailing EPS 하한 | 0.5 | 턴어라운드 제외 목적, KR 단위 조정 |
| gap 상한 | 10배 | 투기 tail 제외 |
| 유동성 | $1B | KR 거래대금 기준 (예: ≥50억/100억) |
| 리밸 | 월 1회 | 동일 |
| 진입/이탈/슬롯 | 7/7/7 strict (gap=신선도라 버퍼 X) | 동일 컨셉 |
| 국면 | 약세장 → 현금 | KR 국면(MA20/80)으로 |

## 5. ★KR이 반드시 주의할 점 (US 시행착오에서 나온 교훈)
1. **반드시 FORWARD 신호로 검증할 것 — 후행(실현)성장으로 프록시하면 가치가 거짓으로 진다.**
   US에서 내가 8년 BT를 **후행gap(작년실적/재작년)**으로 돌렸더니 가치가 모멘텀에 압도당함. **forward gap(컨센 선행 / 실제 다음해 실적)으로 정정하니 Calmar 0.7→2.0으로 역전, 2022 디레이팅 약세장도 방어.** KR도 컨센 forward로 측정해야 진짜 성과가 나옴.
2. **컨센 커버리지 편향**: FnGuide 컨센은 대형·중형 위주 → sleeve 유니버스가 **모멘텀의 broad ~1900보다 좁아짐**(소형 성장주 제주반도체류 미커버). 이건 KR이 이미 아는 함정(커버 안 되는 중소형 제외 시 Calmar 붕괴). sleeve는 "커버되는 대형·중형 가치 바스켓"으로 한정됨을 인지.
3. **look-ahead 한계**: 컨센 forward는 과거 히스토리가 없어 BT가 look-ahead proxy 상한 — KR의 기존 확신가중 노트와 동일 한계. **노이즈 주입 robust 테스트**(US가 함: ±40% 애널오차 줘도 Calmar 1.77 유지)로 보완 권장.

## 6. 검증 체크리스트 (KR 규율대로)
- [ ] **FORWARD 컨센으로** gap 계산 (후행 금지 — 위 5-1)
- [ ] 파라미터 robust (K 스윕 plateau)
- [ ] **LOWO** (슈퍼위너 1~2종목 빼도 견고한가 — US는 tight 스크린이 2종목 베팅이라 붕괴, 넓혀야 robust)
- [ ] walk-forward / 기간분할 (특히 약세장 방어)
- [ ] **★결합 테스트**: 모멘텀 3슬롯 + 가치 sleeve 포트폴리오 Calmar/MDD가 **각각보다 나은가** (= 진짜 시너지 확인. US는 corr +0.83이라 시너지 0이었지만 KR은 +0.015라 여기가 핵심)
- [ ] 메타배분: sleeve를 KR sleeve 내 몇 %로 (현금버퍼·KR/US 비중과 정합)

## 7. 결론
- **US**: gap·모멘텀 상관 높아 시너지 0 → sleeve는 페이퍼 관찰만, 메인은 모멘텀 2슬롯.
- **KR**: gap·모멘텀 직교 → **별도 가치 sleeve가 시너지 낼 구조적 조건을 갖춤.** 검증 통과하면 KR 포트에 "모멘텀(broad 회전) + 가치(대형 고gap 분산)" 2-sleeve가 실제 MDD↓·Calmar↑ 낼 수 있음.
- 단 KR 확신가중(sizing)과 별도 sleeve(새 종목)는 **다른 것** — 결합 테스트로 어느 쪽이/둘 다가 나은지 판정 필요.
